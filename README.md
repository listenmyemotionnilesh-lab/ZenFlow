# Zen Flow

> A minimal, beautiful productivity app designed for focused students and NEET aspirants.

## Philosophy

Zen Flow believes productivity should feel calm, not chaotic. No overwhelming features. No complicated workflows. Just you, your goals, and gentle progress tracking.

## Features

### Smart Sections
- Study, Work, Meditation, Health, Personal sections
- Custom sections with colors and icons
- Drag & drop reordering
- Motivational quotes per section

### Task Management
- Quick add tasks with priorities
- Add completed tasks retroactively
- Notes, reminders, and repeat options
- Swipe gestures and search

### Focus Timer
- Pomodoro (25 min)
- Meditation (15 min)
- Deep Work (60 min)
- Custom sessions
- Beautiful circular progress

### Progress Tracking
- Daily productivity score
- Streak system
- Weekly bar charts
- Study & meditation time tracking

### Design
- Material Design 3
- AMOLED dark mode
- Glassmorphism effects
- Smooth animations
- Premium feel on any device

### Offline & Lightweight
- Works fully offline
- No login required
- Optimized for low-end phones
- SD card install support
- Export/import JSON backups

- **DI**: Hilt
